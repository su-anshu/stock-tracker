# Quick Manual Setup Commands

## Option A: Install Only Essential Packages
```bash
pip install streamlit pandas plotly
```

## Option B: Install from requirements (if needed)
```bash
cd "C:\Users\LENOVO\Desktop\Claude\stock_tracker"
pip install -r requirements.txt
```

## Option C: Run App Directly (if packages already installed)
```bash
cd "C:\Users\LENOVO\Desktop\Claude\stock_tracker"
streamlit run app.py
```

## Speed Tips:
- Use `pip install --no-cache-dir` if installation is slow
- Close other programs to free up system resources
- Use `pip install --upgrade pip` to update pip first

## If Packages Are Already Installed:
Just run: `streamlit run app.py`
